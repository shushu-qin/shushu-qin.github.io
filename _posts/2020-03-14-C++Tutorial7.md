## 1 临时对象的概念

```c++
//i++ ++i
int i = 1;
int &&r1 = i++; //r1和i之间没有什么关系
r1 = 110;
i = 23;
```

另外一些临时对象时因为我们代码的书写问题产生的，统一称临时变量为临时对象。

- 堆：放new生成的变量 

- 栈：一般会存放临时变量

## 2 产生临时对象的情况和解决

class CTempValue

class CTempValue

  //拷贝赋值运算符

    if(this = &other)

Double函数引起的消耗

```c++
CTempValue Double(CTempValue &t)
{
  CTempValue temp;  //会消耗一个构造函数和一个析构函数
  t.val1 = t.val1*2;
  t.val2 = t.val2*2;
  return temp;  //产生临时对象，占用一个拷贝构造函数和析构函数
}
```

如何优化Double

```c++
CTempValue Double(CTempValue &t)
{
  return CTempValue(t.val1*2, t.val2*2);
}

Double(sum); //系统调用了构造函数和1个析构函数
CTempValue t2 = Double(sum); //系统调用了构造函数，没有调用析构函数
//这种优化节省了一个拷贝构造函数和一个析构函数
```
