## 1 位运算

### 1.1 位的概念

一个int型数据占4个字节，一个char类型数据占1个字节内存。

```c
int main()
{
  char abc = 'A';
  // 一个字节由8个二进制位组成。最左边的称为最高位，最右边的称呼为最低位
  // 00000000(0) - 11111111(255)
  int abcd = -189; // 占4个字节
  unsigned int abcd = 189;
  // 11111111,11111111,11111111,11111111 = 4,294,967,295
  abcd = 4,294,967,297; //溢出 4,294,967,297-4,294,967,296  
}
```

### 1.2 位运算符

肯定是针对二进制位进行运算的

| 位运算符 |          |
| -------- | -------- |
| &        | AND      |
| \|       | OR       |
| ^        | 按位异或 |
| ~        | 取反     |
| <<       | 左移     |
| >>       | 右移     |

```c
// 相当于逻辑运算符，0为False，1为True
// AND
// 0&0 = 0 0&1 =0 1&1 =1 有0出0，全1出1 
unsigned int tempvalue = 38&22;
// 100110  38
// 010110  22
//--------
// 000110  6

// OR
// 有1出1
unsigned int tempvalue = 38|22;
// 100110  38
// 010110  22
//--------
// 110110  54

// NOR
// 全一样出0
unsigned int tempvalue = 38^22;
// 100110  38
// 010110  22
//--------
// 110000  54

// 0111 1010 跟1做亦或，就能翻转某些位，若想保留，就和0做亦或

// << 相当于原来的十进制数乘以2
unsigned int a = 15 << 1;  // 30

// >> 相当于原来的十进制数除以2
unsigned int a = 15 >> 2;  // 7 -> 3

a & = b;  // a = a&b
```

## 2 位运算的应用

```c
#define BIT(x) (1<<(x))  // BIT(0) ==> 1左移0位
												 // BIT(1) ==> 1左移1位  2
int main()
{
  // int task[10]  //太占内存
  // (1) 每日任务有10个
  // (2) 只需记录任务是否完成，0表示没完成，1表示完成
  // unsigned int task; // 4个字节，32位
  int i;
  
  // 如果想提取第7位，只需 task & 1000000
  // 第7位为0， task&1000000 = 0
  // 第7位为1， task&1000000 = BIT(6)
  
  // 10个任务
  enum EnumTask
  {
    ETask1 = BIT(0),
    ETask2 = BIT(1),
    //...
    ETask10 = BIT(9)
  }
  
  unsigned int task = 0;
  if(task&ETask7)  // 提取第7位
  {
    //任务7已做过
	}
  else
  {
    //任务7没做过
  }
  
  // 如果想把第7位设为1，只需 task | 1000000
  // 第7位为0， task&1000000 = 1
  // 第7位为1， task&1000000 = 1
  
  // 把任务5做了
  task = task | Etask5;
}

```
