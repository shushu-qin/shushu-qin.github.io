## 3. 函数

### 3.1 函数调用方式

- a)把函数作为一个语句

```c
printhello();
```

- b)函数出现在一个表达式中，这种表达式成为函数表达式，这个时候要求函数返回一个确定的值参加表达式运算

```c
result = max(12,10)*10
```

c语言中不允许在函数中定义另外一个函数，不允许函数的嵌套定义。但允许函数的嵌套调用。

- c)函数递归调用

#### 递归调用的定义

```c
void fun1()
{
	printf("递归函数执行\n");
	fun1();
}


int main()
{
	fun1();
	return 0;
}
```

#### 递归调用的出口

范例：计算5的阶乘

a)我们不知道5的阶乘，但我们知道4的阶乘*5 = 5的阶乘

b)以此类推。。。

c)我们不知道2的阶乘，但我们知道1的阶乘*2 = 2的阶乘

d)1的阶乘为1

递归调用的出口肯定是在1的阶乘

````c
int f(int n)
{
  int result;
	if(n ==1 )
  {
    return 1;
  }
  else
  {
    result = result * f(n-1);
 	  return result;
  }  
}
````

优点：代码少，简洁

缺点：不好理解；如果调用太深，调用栈（内存）可能会溢出；效率和性能都不高

有些问题用不用递归都行。有些问题则必须用递归解决，例如，汉诺塔。

### 4. 局部变量和全局变量

#### 4.1 变量的存储类别

```
-----------
|程序代码区 |
-----------
|静态存储区 |
-----------
|动态存储区 |
-----------
```

从变量存在的时间（生存期）角度划分，我们可以把变量分为：静态存储变量和动态存储变量。

- 静态存储变量：在程序运行期间分配固定存储空间的变量

  - 全局变量：在函数外部定义的，放在`静态存储区`中，程序开始执行的时候就给全局变量分配存储区，程序执行完毕后就释放这些存储区。在程序执行过程中他们占据固定存储单元。

- 动态存储变量：在程序运行期间根据需要动态分配存储空间的变量

  动态存储区中有哪些数据？

  - 函数形参：函数形参被看做局部变量

  - 函数内定义的一些变量

  - 函数调用时现场的一些数据和返回地址等

    - 一般来讲，这些数据在函数调用开始时分配存储空间，函数调用完毕这些空间就被释放掉了，这种分配和释放，我们就认为是`动态`的。

      如果你两次调用同一个函数，分配给此函数的局部变量等存储空间地址是不同的。

#### 4.2 特殊情况：局部静态变量

```c
void fun1()
{
  int c = 4;
  printf("c = %d \n", c);
  c++;
}

void fun1()
{
  static int c = 4;
  printf("c = %d \n", c);
  c++;
}
```

- 在静态存储区分配存储单元，程序整个运行期间不释放
- 局部静态变量是在编译时赋予初值的，只赋予一次。在程序运行时，它已经有了初值，以后每次调用函数时，不再重新赋初值，而是保留上次函数调用结束时的值。
- 虽然局部静态变量里的函数在函数调用后存在，但其他函数不能引用它。
- 缺点：长期占用内存，降低了程序的可读性。

结论：除非必要，否则不要多用局部变量。

#### 4.3 全局变量跨文件引用

当我们在一个文件中定义了全局变量，在其他文件中想要使用它是，可以用`extern` 做一个外部变量说明，表示这里出现的变量是一个已经在其他文件中定义过的外部变量，本文件不必为它分配内存。

```c
// project1.cpp
#include <stdio.h>
extern int g_a;  
extern fun1();
//必须放在所有函数开头，不能放在函数里面
int main()
{
  printf("g_a = %d", g_a);
  // .....
}
```

```c
// project2.cpp
int g_a = 6
void fun1()
{
  // ....
  return;
}
```

静态全局变量：在定义全局变量前加`static`，则该全局变量只能在本文件中使用。

```c
// project2.cpp
static int g_a = 6 ;
void fun1()
{
  // ....
  return;
}
```

#### 4.4 函数的跨文件调用

内部函数：

​				  只能在本文件中使用

​				   static 类型标识符 函数名(形参表) {..}

外部函数：

​					如果一个函数定义不用static，它就是外部函数。extern可加可不加。

​					其他文件，调用外部函数时必须加extern。
