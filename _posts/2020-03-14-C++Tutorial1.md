## 1 范围for语句

用于遍历一个序列。

```c++
int main()
{
  int v[]{1,2,3,4};
  for(auto x:v)
  {
    cout << x << endl;
  }
  
  for(auto &x:v)  // 省了拷贝这个动作，提高了系统效率
  {
    cout << x << endl;
  }
  /*
  for(auto x:{1,4,5})
  {
    cout << x << endl;
  }
  */
	return 0;
}
```

## 2 动态内存分配问题

c语言中供程序使用的内存空间：

- 程序区
- 静态存储区：全局变量……
- 动态存储区：局部变量……

c++中供程序使用的内存空间，进一步分为五个区：

- **栈**：一般函数内的局部变量

  ```c++
  void func(){
  	int abc;
  }
  ```

- **堆**：程序员用malloc/new分配，用free/delete来释放。忘记释放后，程序运行结束时，系统会回收。

- 全局/静态存储区：放全局变量和静态变量static

- 常量存储区："Hello world"

- 程序代码区

堆和栈的区别：

- 栈：空间有限，分配速度快

- 堆：只要不超过你实际拥有的物理内存，也在操作系统允许你能分配的最大内存大小之内，都可以分配给你。分配速度比栈慢。你可以随时用new/malloc来分配，free/delete释放，非常灵活。

  - 在c语言中用malloc/free，它们是函数。

    memory allocation：动态内存分配

    一般形式： `void *malloc(int NumBytes);` NumBytes：要分配的内存数。分配成功，则返回指向分配内存的指针，失败则返回NULL。当这段分配内存不使用，应该用free()释放，供其他地方使用。

    free：`void free(void *FirstByte)`

    ```c++
    // malloc free
    int *p = NULL;
    p = malloc(sizeof(int));
    if(p !=  NULL)
    {
      // 分配成功
      *p = 5;
      cout << *p << endl;
      free(p);
    }
    
    char *pc = NULL;
    pc = (char*) malloc(100*sizeof(char));
    if(pc != NULL)
    {
      //strcpy(pc,"hello world!");  // 如果超过pc空间数，不会报错
      strcpy_s(pc,100，"hello world!");  // 如果超过pc空间数，会报错
      free(pc);
    }
    
    int *p = (int *)malloc(sizeof(int)*100); // 分配100个可以放下整数的内存空间
    if(p != NULL)
    {
    	int *q = p;
      *q++ = 1; //==>*(q++): *q = 1; q++
      cout << *p << endl;
      cout << *(p+1) << endl;
      free(p);
    }
    ```

    ```c++
    // new/delete c++中就用new/delete,不再使用malloc和free。它们还干了更多的事。
    
    // new 一般使用格式
    // (1) 指针变量名 = new 类型标识符
    // (2) 指针变量名 = new 类型标识符(初始值)
    // (3) 指针变量名 = new 类型标识符[内存单元个数]
    
    int *myint = new int; // int *p = (int *)malloc(sizeof(int))
    ```

## 3 nullptr: c++11中引入的新关键字

nullptr代表的也是空指针。

```c++
//char *p = NULL; //NULL实际就是0
char *q = nullptr;
int *p = nullptr;
//int a = nullptr; //fail
int b = NULL; //succeed
if(p == nullptr)
{
	cout << "good" << endl;
}
if(q == NULL)
{
  cout << "good" << endl;
}
//主要是为了避免指针和整数之间的混淆
cout << typeid(NULL).name() << endl;
cout << typeid(nullptr).name() << endl;
// nullptr和NULL实际上是不同的类型

```

结论：对于指针的初始化，和以往在C语言中用到和指针有关的NULL的场合，能用nullptr的全用nullptr取代NULL

```c++
int *pa = new int[100];
if ( pa!= nullptr )
{
  int *q = pa;
  *q++ = 12; //[0] = 12
  *q++ = 18; //[1] = 18
  cout << *pa << endl;
  delete[] pa; //释放int数组空间
}
```
